**Ledger Finance** is an enterprise-grade, AI-powered financial management platform purpose-built for individuals, professionals, and families who want more than just transaction logs.

The platform leverages **Retrieval-Augmented Generation (RAG)**, **vector embeddings**, and **agentic AI pipelines** to transform raw financial data into actionable intelligence — surfacing insights, optimizing tax strategies, and guiding investment decisions in real time.

Unlike conventional finance apps that passively record data, Ledger Finance **thinks with you** — analyzing patterns, flagging anomalies, recommending actions, and growing smarter with every transaction.

| Dimension | What Ledger Finance Does |
|---|---|
| 💡 Intelligence | AI-driven financial insights from your actual data |
| 🤖 Automation | Automated bill scanning, SMS parsing, and categorization |
| 📊 Analytics | Deep financial reporting across all categories |
| 🏦 Investment | Portfolio building, tracking, and AI-powered suggestions |
| 🧾 Tax | Personalized tax-saving recommendations in real time |
| 👨‍👩‍👧 Family | Role-based collaborative household financial management |
| 🎮 Engagement | Gamified financial discipline with streaks, XP, and challenges |

## ⚡ Installation Guide

### Prerequisites

```bash
node >= 20.0.0
npm >= 10.0.0
MongoDB >= 7.0
Git
```

### 1. Clone the Repository

```bash
git clone https://github.com/ledger-finance/ledger.git
cd ledger
```

### 2. Environment Configuration

```bash
# Backend environment
cp backend/.env.example backend/.env
```

```env
# backend/.env

# Server
PORT=5000
NODE_ENV=development

# Database
MONGODB_URI=mongodb://localhost:27017/ledger_finance

# Authentication
JWT_SECRET=your_super_secure_jwt_secret_here
JWT_EXPIRES_IN=7d

# AI Configuration
GEMINI_API_KEY=your_google_gemini_api_key
VECTOR_DB_URL=your_vector_database_url

# File Storage
CLOUDINARY_URL=your_cloudinary_url
```

```bash
# Frontend environment
cp frontend/.env.example frontend/.env
```

```env
# frontend/.env
VITE_API_BASE_URL=http://localhost:5000/api/v1
VITE_APP_NAME=Ledger Finance
```

### 3. Install Dependencies

```bash
# Install backend dependencies
cd backend && npm install

# Install frontend dependencies
cd ../frontend && npm install
```

### 4. Database Setup

```bash
# Seed initial data (optional)
cd backend
npm run db:seed
```

### 5. Start Development Servers

```bash
# Start backend server (from /backend)
npm run dev

# Start frontend server (from /frontend)
npm run dev
```

### 6. Access the Platform

```
Frontend  →  http://localhost:5173
Backend   →  http://localhost:5000
API Docs  →  http://localhost:5000/api/docs
```

### Docker Setup (Recommended for Production)

```bash
# Build and launch all services
docker-compose up --build

# Run in detached mode
docker-compose up -d
```

1. **Register** an account at `/auth/register`

## 🤖 AI System Design

<div align="center">

### RAG Financial Intelligence Pipeline

</div>

```
┌─────────────────────────────────────────────────────────────────────┐
│                    RAG PIPELINE ARCHITECTURE                        │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  USER QUERY: "How can I reduce tax this year?"                      │
│       │                                                             │
│       ▼                                                             │
│  ┌────────────────────────────┐                                     │
│  │   EMBEDDING MODEL          │  ← Query → Vector Representation   │
│  └────────────────┬───────────┘                                     │
│                   │                                                 │
│                   ▼                                                 │
│  ┌────────────────────────────┐                                     │

│  │   · Transaction History    │                                     │

│  │   · Investment Data        │                                     │
│  └────────────────┬───────────┘                                     │
│                   │                                                 │
│                   ▼                                                 │
│  ┌────────────────────────────┐                                     │
│  │   CONTEXT AUGMENTATION     │  ← Merge retrieved data + query    │
│  └────────────────┬───────────┘                                     │
│                   │                                                 │
│                   ▼                                                 │
│  ┌────────────────────────────┐                                     │
│  │   GEMINI LLM REASONING     │  ← Generate grounded response      │
│  └────────────────┬───────────┘                                     │
│                   │                                                 │
│                   ▼                                                 │
│  RESPONSE: "Based on your ₹45,000 in eligible investments,         │
│  you can save ₹13,500 under Section 80C. Consider increasing        │
│  your ELSS allocation by ₹20,000 before March 31."                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 🔌 API Overview

#### Authentication
| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/auth/register` | Register new user |
| `POST` | `/auth/login` | User login |
| `POST` | `/auth/refresh` | Refresh JWT token |
| `POST` | `/auth/logout` | Invalidate session |

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/transactions` | Get all transactions |
| `POST` | `/transactions` | Create transaction |
| `PUT` | `/transactions/:id` | Update transaction |
| `DELETE` | `/transactions/:id` | Delete transaction |
| `POST` | `/transactions/scan` | AI bill scan upload |

#### AI Advisor
| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/ai/chat` | Send financial query |
| `GET` | `/ai/insights` | Get proactive insights |

| Method | Endpoint | Description |
|---|---|---|

#### Family Finance
